== Description ==

El plugin Banner System permite vender 5 diferentes espacios publicitarios de banners 350x250, 728x90, 300x600, 234x60, 320x100 y permite llevar un registro de clicks dados y contar los días que le restan al banner hasta cuando vuelve estar disponible.
Actualmente como medio de pago esta integrado flow de Chile.

Si desea integrar nuevo medio de pago pongase en contacto con el desarrollador.


== Installation ==


1. Descarga el plugin
2. Ingresa al administrador de tu wordPress.
3. Ingresa a Plugins / Añadir-Nuevo / Subir-Plugin.
4. Busca el plugin descargado en tu equipo y súbelo como cualquier otro archivo.
5. Después de instalar el .zip lo puedes ver en la lista de plugins instalados , puedes activarlo o desactivarlo.


== Frequently Asked Questions ==

= ¿ Como se muestran los banners ? =

Los banners se muestran atrábves del uso de shortcodes, los cuales debera simplemente colocar donde quiera que se muestre.

= ¿ Tamaños de banners ? =

350x250, 728x90, 300x600, 234x60 y 320x100

= ¿ Formatos de imágenes permitidos ? =

jpg, png y gif

= ¿ El cliente cuenta con zona de miembros ? =

Si, el cliente además de gestionar, puede recibir estadisticas de clicks realizado y saber cuantos días le restan a cada banner.


= ¿ Qué más debo tener en cuenta, que no me hayas dicho ? =

1. Recuerde que el medio de pago actual es flow de chile, pero podra pedir que se añade uno nuevo.
2. El formulario de pago es muy simple y carece de estilo puede añadirle unas cuantas reglas de css a su preferencia.
